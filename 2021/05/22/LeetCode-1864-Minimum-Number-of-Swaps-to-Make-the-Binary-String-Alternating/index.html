LeetCode 1864. Minimum Number of Swaps to Make the Binary String Alternating

https://leetcode.com/problems/minimum-number-of-swaps-to-make-the-binary-string-alternating/

比賽的時候看到input規模以為可以寫個O(n^2)的算法，後來花了一些時間還是覺得不太好寫才突然靈光一現想到O(n)的寫法，但是那個時候求快吃了一次WA，現在再回來發現了別人寫的更周全簡潔的寫法！

這題的突破口在於有沒有意識到題目要求的alternating條件，其實只有兩種情況，要嘛1開頭不然就是0開頭；而且當長度不為even的話那就是以出現次數多的那個數字當開頭，就這樣。剩下就只需要將有不同的地方累加起來最後再/2即可。

另外要注意的是當1跟0出現的次數落差>1，那當然就不可能透過swap任意兩個位置的char使string變成alternating囉。然後模擬string 01變化的寫法不要忘啊>< 真的好用！
